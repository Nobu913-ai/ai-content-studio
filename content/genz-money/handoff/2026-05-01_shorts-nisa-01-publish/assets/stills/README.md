# サムネイル候補 (未生成)

YouTube Shorts は動画の最初フレームが自動的にサムネ化されるため、
通常はカスタムサムネは不要。

ただし、長尺ビデオ版を作成する場合や、Instagram カルーセル投稿時は
カスタム画像が有効。

## 推奨サムネ視覚案 (seo.json thumbnail_ideas より)

1. 「100万円→80万円？」 (本命) — 動画 08-01 最終比較コマと連続
2. 「20万円の差」 — 落差を強調
3. 「税金0円」 — NISA カードの緑グロー
4. 「100円から」 — 低ハードル訴求
5. 「360万円・無期限」 — 包括感

カスタムサムネを作る場合は Canva 等で 1280x720 (横) または 1080x1920 (縦) で書き出し、
このディレクトリに配置すること。
