# コンプライアンス・品質チェック

台本の品質とコンプライアンスを自動監査し、人間が編集すべきポイントを特定します。

## 手順

1. `$ARGUMENTS` で台本パスが指定されていない場合、最新の台本を探す:

```bash
ls -t content/*/scripts/*.md | head -5
```

2. コンプライアンスチェックを実行:

```bash
cd ai-content-studio && node src/cli.js check $ARGUMENTS
```

3. 生成されたレポートを読み込み、以下を報告:

### スコア概要
- 総合スコアと判定（pass/warn/fail）
- 各カテゴリのスコア

### 重要な指摘事項
- **誇大表現**: 「必ず」「絶対」「誰でも」等の検出
- **AI臭さ**: テンプレ的な表現、具体性不足の検出
- **金融コンプライアンス**: 免責事項、情報源、投資推奨との区別
- **独自性**: 過去動画との重複感

### 人間が編集すべきポイント（最重要）
- どこに独自エピソード・具体例を追加すべきか
- テンプレ感がある箇所の書き直し提案
- 人間の温度感が必要な箇所

## 注意事項
- **genz-money（金融系）は毎回必ず実行すること**
- fail 判定の台本はそのまま投稿しない
- warn は指摘箇所を修正してから投稿
