# 収益化レポート

全チャンネルの収益化戦略と現在のセットアップ状況を確認します。

## 手順

1. まず全体のロードマップを表示:

```bash
cd ai-content-studio && node src/cli.js monetize
```

2. `$ARGUMENTS` でチャンネルが指定されている場合、そのチャンネルの詳細:

```bash
cd ai-content-studio && node src/cli.js monetize $ARGUMENTS
```

3. `config/monetization.js` を読み込み、以下をチェック:
   - アフィリエイトの URL が設定されているか（空欄 = 未登録）
   - デジタル商品の URL が設定されているか
   - 未設定の項目があれば、登録を促す

4. 収益化の現状を診断し、以下を報告:

### 今すぐやるべきこと（未設定の項目）
- ASP（A8.net等）への登録状況
- アフィリエイトリンクの発行状況
- デジタル商品の作成・販売ページ状況

### 月次目標の達成に必要なアクション
- Month 1 目標: ¥5,000 → 必要なアフィリエイト成約数
- Month 2 目標: ¥20,000 → 必要なアフィリエイト成約数 + デジタル商品販売数
- Month 3 目標: ¥50,000 → 必要な成約数 + 販売数 + スポンサー案件数

### 収益源の優先順位
1. **genz-money のアフィリエイト**が最速。SBI証券/楽天証券の口座開設で CPA ¥2,000-5,000
2. **デジタル商品**（Notionテンプレート、PDF）は初月から販売可能
3. **広告収益**は YouTube 収益化条件達成後（3-6ヶ月後）
