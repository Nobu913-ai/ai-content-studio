# 動画アイデア ブレインストーミング

チャンネルの方向性とターゲット層に合った新しい動画アイデアを生成します。

## 手順

1. `$ARGUMENTS` でチャンネルが指定されていればそのチャンネル、なければ全チャンネル対象
2. `config/channels.js` を読み、既存トピックを確認する（重複を避けるため）
3. 以下の**5つの切り口**からそれぞれ2-3個ずつアイデアを出す:

### 切り口

| # | 切り口 | 説明 |
|---|--------|------|
| 1 | **トレンド連動** | 現在話題のニュース・イベントに絡めたネタ |
| 2 | **既存ヒットの派生** | 人気が出そうな既存トピックの続編・深掘り |
| 3 | **対決・比較系** | 「A vs B」形式で視聴者の関心を引くネタ |
| 4 | **ランキング・リスト系** | 「○○ベスト5」「知らないと損する○○3選」 |
| 5 | **反常識・意外性** | 「実は○○は間違い」「○○の本当の意味」 |

4. 各アイデアについて以下を提示する:

```
### アイデア: <タイトル案>
- 切り口: <上記5つのどれか>
- 推定検索ボリューム: 高/中/低
- バズ可能性: 高/中/低
- 制作難易度: 低/中/高
- ショート展開: <30秒に切り出せるフック>
```

5. ユーザーが気に入ったアイデアがあれば `/add-topic` でトピック登録するか、`/full-pipeline` で即座に台本生成に進むか提案する

## チャンネル別のアイデア方針

- **japanese-mindset**: 英語圏で「日本」に関連して検索されるキーワードを意識。Quora、Reddit の Japan 関連スレッドで話題になるテーマ。
- **genz-money**: 給料日、ボーナス時期、確定申告シーズンなど時期に連動したネタ。TikTok の金融インフルエンサーが扱っているテーマ。
- **chill-culture**: 季節感のあるネタ（夏→冷房カフェ、冬→サウナ）。Instagram で「#チル」「#整う」で投稿されているトレンド。
