# Shorts先行仮説検証

トピックから直接Shorts台本3本を生成します。長尺動画を作る前に、異なる切り口でテストし、反応が良いテーマのみを長尺化する戦略です。

## 手順

1. Shorts先行テストを実行:

```bash
cd ai-content-studio && node src/cli.js shorts-first $ARGUMENTS
```

2. 生成された3本の Shorts について以下を報告:
   - 各Shortの切り口（angle）
   - フック文（最初の1-2秒のセリフ）
   - 拡散可能性スコアと理由
   - 長尺化のポテンシャル

3. 投稿戦略を提案:
   - どのプラットフォームに先に投稿するか
   - 投稿のタイミング
   - ハッシュタグ戦略

## 運用フロー

```
shorts-first でテスト投稿
→ 1-2週間反応を見る
→ 勝ちテーマ: pipeline で長尺化
→ 負けテーマ: 別の切り口でリトライ or 次のトピックへ
```

## 判断基準（投稿後）
- **長尺化GO**: 再生数 1,000+、視聴維持率 50%+、コメント率 3%+
- **リトライ**: 再生数 500-1,000、切り口を変えてもう一度
- **スキップ**: 再生数 500未満、次のトピックへ
